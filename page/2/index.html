写在前面

发表于 2018-01-04 | | 阅读次数:

    我也不知道自己为什么会选择这个职业 (血坑)。小时候立志成为一名解放军，一心想把自己奉献给人民与祖国，学生时代想成为一个建筑师，所以高考完了，第一志愿果断填了土木。然而……现实往往是背道而驰的。所以我不出意外的逃了四年课，顺理成章的毕了业。有时候也会抱怨，有时候也会意淫一些牛逼哄哄拯救世界的剧情，每天就这么浑浑噩噩活。最近才开始折腾github，想在自己撸码的时候，记录一下自己的坑和填坑的心得。还是希望自己能继续下去吧，因为我似乎没什么事坚持下来了。引用某个前辈说过的一句话，也许是因为天真到认为自己可以改变世界，所以我才选择了做一只猴。
