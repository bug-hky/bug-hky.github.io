element-ui的render-header属性

发表于 2018-01-29 | 阅读次数:

    这一阵子使用element-ui的时候，一直苦恼于它table组件里的表头的改造。好不容易把样式重置了一遍，现在有需要加N多自定义的样式，类似于表头的筛选，下拉之类的都比较常规，官方文档上面都有详细的教程。那么如果想要自定义表头该怎么办?

    也是束手无策了一阵子，终于在github上找到了答案 => Render Functions，render函数。平时写组件都是一个套路，用template标签构造模板，再在里面做文章。表头支持headerRender属性来自定义表头。
1.编写headerRender函数(用renderHeader属性创建DOM节点，用来渲染自定义表格头部，将函数作为同名属性传入需要自定义的列el-table-column)

2.给需要自定义表头额的列的标签加上属性header-render，可选的为添加的DOM添加CSS样式

这样一个效果大概就可以撸出来了。
