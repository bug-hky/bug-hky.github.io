scss的使用总结

发表于 2018-03-12 | | 阅读次数:

    如今业界的CSS大神还是非常多的，逛掘金，看简书都能时不时的看到一篇优雅的讲述css奇淫巧技的干货，就像捡到了八倍镜和马格南。我就想在填坑的同时总结一些用法，不定时更新一下，方便自己查了。

布局:

元素等间距排放

相同样式的元素水平布局，相邻元素间间隔已知。calc()函数支持基本运算，可以单一或混合使用。（加
减号前后需要加上空格，乘除号后面不需要）

scss

width: calc((100% - 已知间隔 * (元素个数 - 1)) / 元素个数);

less

calc(~"100% - 已知间隔")

元素水平居中

display: inline-block + text-align: center

.parent{
  display: inline-block;
}
.child{
  text-align: center;
}
=>兼容ie与主流浏览器，但是子元素的内容也会居中

display: inline-block + margin:0 auto

.parent{
  display: inline-block;
}
.child{
  margin: 0 auto;
}
=>兼容ie8+，但是div <=> table

display: table + margin:0 auto

.parent{
}
.child{
  display:table;
  margin:0 auto;
}
=>兼容ie8+，但是div <=> table

position: absolute + transform

.parent{
  position: relative;
}
.child{
  position: absolute;
  left: 50%;
  transform:translateX(-50%);
}
=>兼容性高版本浏览器要加前缀，但是这种不影响其他元素

position: flex + margin

.parent{
  display: flex;
}
.child{
  margin: 0 auto;
}
=>兼容性高版本浏览器要加前缀，但是这种不影响其他元素

弹性布局

display: flex;
flex-flow: row/column nowrap;
justify-contnet: felx-start/flex-end/center/space-between/space-around;
align-items: flex-start/flex-end/center

栅格布局

触发方式

行内元素 display: grid;
块元素 display: inline-grid;

grid容器属性

grid-template-columns
grid-template-rows

grid子项属性

grid-column-start: 生效样式的;
对于栅格布局理解其实不是一件难事，关键在于运用，由于属性名称较多，需要经常性的是用来熟悉它。

元素:

单位

px/em

基操勿6

rem

我们都比较熟悉，它是通过用js动态监听dom的状态来改变dom的根fontSize属性
(function (doc, win) {
  let docEl = doc.documentElement
  let resizeEvt = 'orientationchange' in window ? 'orientationchange' : 'resize'
  let recalc = function () {
    var clientWidth = docEl.clientWidth
    if (!clientWidth) return
    docEl.style.fontSize = 14 * (clientWidth / 320) + 'px'
  }
  if (!doc.addEventListener) return
  win.addEventListener(resizeEvt, recalc, false)
  doc.addEventListener('DOMContentLoaded', recalc, false)
})(document, window)

vw/vh

vh/vw是属于css3的单位，与JS没有交集，vh/vw分别是对应视口的高度与宽度的十分之一，所以这个单位
的大小是相对与视口而变化的，是一个相对单位。视口就是设备的可视区域。由于与js没有耦合，所以不会
损耗js的性能。1vh/vw分别是视口高宽的十分之一，就是将设备的高宽进行十等分，在不同设备上的尺寸的
高宽的真实长度是不一致的。

背景

gradient渐变

线性渐变

/* 渐变轴为45度，从蓝色渐变到红色 */
linear-gradient(45deg, blue, red);

/* 从右下到左上、从蓝色渐变到红色 */
linear-gradient(to left top, blue, red);

/* 从下到上，从蓝色开始渐变、到高度40%位置是绿色渐变开始、最后以红色结束 */
linear-gradient(0deg, blue, green 40%, red);

径向渐变

background-image: radial-gradient(circle | ellipse at 圆心位置（% %）,
color 径向对应颜色位置, color 径向对应颜色位置, ...);
background-color: color;

重复渐变

.linear-repeat {
  background: repeating-linear-gradient(to top left, lightpink, lightpink 5px, white 5px, white 10px);
}

.radial-repeat {
  background: repeating-radial-gradient(powderblue, powderblue 8px, white 8px, white 16px);
}
