迭代

for ... in ...

list

L = [1, 2, 3]
for x in L:
  print x
=> 1 2 3

dict

D = {'y': 6, 'x': 5, 'z': 7}
for x in D:
    print(x)
=> 'y' 'x' 'z'
for x in D.values():
  print(x)
=> 6 5 7
for x, y in D.items():
  print(x, y)
=> 'y', 6 'x', 5 'z', 7

生成器 generator

1.把生成式的[]改为()就是一个generator

2.如果一个函数定义中包含yield关键字，那么这个函数就不再是一个普通函数，而是一个generator

def fib(max):
    n, a, b = 0, 0, 1
    while n < max:
        yield b
        a, b = b, a + b
        n = n + 1
    return 'done'
