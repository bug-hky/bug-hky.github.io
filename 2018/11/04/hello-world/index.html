本地项目push操作同步到coding.net与github上

发表于 2018-11-04 | 阅读次数:

原因

    自己想做点demo练手的时候，总是需要github和gitlab之类的托管所。我想把commit了的代码push到两个远程仓库上，又不想push两次。下面是拿github 和 coding.net举例，所以：

1.先查看一下已有的github远程库

git remote -v
origin  https://github.com/user/project.git (fetch)
origin  https://github.com/user/project.git (push)

2.创建一个名为’both’的远程仓库

git remote add both https://git.coding.net/XXX.git

3.给’both’添加push到 coding.net 与 github 仓库的指令

git remote set-url --add --push both https://git.coding.net/XXX.git
git remote set-url --add --push both https://github.com/XXX.git

4.push前下拉代码解决冲突

git pull both master(branch)

5.push both

git pull both 如果成功就能分别看到coding.net 与 github上面的最新的版本号了
